# How to mojo

### Install

Install the Modular CLI:
```
curl https://get.modular.com | \
  MODULAR_AUTH= \
  sh -
```

Install the Mojo SDK:
```
modular install mojo
```


Path
```
echo 'export MODULAR_HOME=""' >> ~/.bashrc
echo 'export PATH=":$PATH"' >> ~/.bashrc
source ~/.bashrc
```
